# Snake game (in C) to work with BeanBoard RA8875 Hardware SPI Example

A simple terminal based snake game written in C, adapted for the BeanBoard RA8875 Hardware SPI Example.

This requires the accompanying beanboard_RA8875_hardware_SPI asm program to be loaded on BeanZee

ra8875.asm maps labels to the beanboard_RA8875_hardware_SPI function addresses in beanboard_RA8875_hardware_SPI.
